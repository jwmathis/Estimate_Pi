About This Webapp

This web application estimates the value of Pi using the Monte Carlo method, a computational algorithm that relies on repeated random sampling to obtain numerical results.

The core idea is to simulate random points within a square that encloses a circle. The ratio of points falling inside the circle to the total number of points is used to approximate the value of Pi. The more points you use, the more accurate the estimate will be.
